Project Documentation

The Challenge

Skin cancer detection relies heavily on identifying microscopic visual patterns. This project bridges the gap between raw medical data and clinical insight using Deep Learning to categorize lesions from the HAM10000 dataset.

Preprocessing Pipeline

Images undergo a specific transformation pipeline: initial resizing to 224x224 pixels, conversion to Tensors, and normalization using ImageNet means ($\mu$) and standard deviations ($\sigma$).

🔬 ABCD Assessment Guide

A

Asymmetry: One half of the lesion does not match the other half.

B

Borders: Edges are irregular, notched, or blurred.

C

Color: Pigmentation is not uniform.

D

Diameter: Lesion is greater than 6mm.
